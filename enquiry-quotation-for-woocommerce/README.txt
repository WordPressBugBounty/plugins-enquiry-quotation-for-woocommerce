Product enquiry for WooCommerce and quote request plugin that can save enquiries and email the WooCommerce product enquiry as well

== Description ==

[Try Pro version front and backend demo](https://websitemaintenanceservice.in/enquiry_demo/) | [Documentation](https://www.piwebsolution.com/user-documentation-product-enquiry-for-woocommerce/)

The **Product Enquiry for WooCommerce plugin** adds an enquiry button to products. Using this, customers can send you an enquiry or request a quote for a product.

Using this Product enquiry for WooCommerce plugin, customers will be able to send aggregated enquiries for multiple products at the same time, as all the enquired products get added into the enquiry cart  

== Product enquiry for WooCommerce plugin features ==

* Show enquiry button on **Shop/Category page** and **Single product page**
* WooCommerce product enquiry plugin creates a complete **cart of inquiry**
* Add multiple products in **enquiry cart**
* Add different **message along with each product** in the enquiry cart
* Receive **inquiry details in the email**
* All the submitted enquiries are **saved in your site**, so you can look up historical enquiries on your website
* Collect **name, email, phone number** along with enquiry
* Change the **Text** shown in the enquiry button
* Change the **background** and text color of the button 
* Have different background and text color for the button on the product loop page and the single product page 
* Option to disable logo in enquiry email
* Enquiry cart page is auto created during installation of the plugin, but if you want you can change the page; you can show enquiry cart on any page using the shortcode **[pisol_enquiry_cart]**. Make sure to select the page in the advanced option as enquiry page
* Pass the enquiry data to a webhook so you can use that data in Zapier and other platforms that support webhooks
* **Honeypot** for spam protection
* **Captcha** for spam protection

== Frequently Asked Questions ==

= Can I only show the product enquiry button on the product loop page (Shop/Category page) =
Yes, there is an option to enable/disable the WooCommerce product enquiry button for the Shop or Category page

= I want to show the product enquiry button on the single product page only =
Yes, there is an option to enable the enquiry button on the single product page

= I want to create a product catalog site using WooCommerce =
This is the perfect plugin for you. Using its pro version you can disable the Add to cart button on all the products and show only the product enquiry button

= I want to show the product enquiry button only for out-of-stock products =
Yes, there is an option for that. Once you enable that option, the enquiry button will be shown only when the product is out of stock

= I installed the plugin but I don't see any inquiry button on variable products =
The free version does not support variation products; you have that option in the pro version

= I want to add multiple emails for the enquiry receiver =
Yes, you can add multiple email IDs separated by commas in the pro version

= I want to check old enquiries =
All the enquiries that you get are saved on your site under the enquiry, so you can check all the old enquiries 

= I want to add more email IDs in the enquiry receiver =
In the pro version you can add more than one email ID to receive the enquiry 

= Want to send a copy of the enquiry to the customer =
In the pro version a copy of the email is sent to the customer as well, and you have the option to disable it

= I want to change the logo shown in the product enquiry email =
Yes, you can change the logo in the pro version

= I don't want to show product price =
Yes, you can hide product price; there is an option for that. Once the product price is hidden they will not have the Add to cart button as well

= I want to hide product price only for Guest customers (users who are not logged in) =
Yes, you can do that

= I want to remove WooCommerce cart and checkout page =
Yes, there is a 301 redirect option. Once you enable this, all the traffic going to Cart and Checkout pages will be redirected to the product enquiry cart page

= I want to add address and phone number and more details in the email sent to the customer =
In the pro version you can add extra content from the plugin backend

= I want to make a form field like subject and message non-required =
You can do that in the Pro version; in that you can make any field non-required

= I want the add to enquiry button to redirect to the enquiry page =
Yes, Pro has the option. By enabling that, the add to enquiry click will redirect to the enquiry page

= I want to disable enquiry for a specific product =
Yes, you can do that in the PRO version. You can disable the inquiry option for a specific product from the "Enquiry option" tab in the product.

= I want to show users how many products are added to their enquiry cart =
Yes, in the pro version you can show a cart icon on all the pages of the site. This icon will show how many products are added to the enquiry cart

= I don't want to show the price on the enquiry cart and enquiry email =
Yes, you can do that in the pro version; it has the option to remove the price columns from the enquiry cart and enquiry email sent 

= I want the customer to accept my site terms and conditions before submitting the enquiry =
In the pro version you get the option to enable the terms and conditions option. Once enabled, the customer has to accept the terms and conditions checkbox; then only their enquiry will be submitted

= I want to insert the inquiry cart icon using shortcode at different locations =
Yes, the Pro version allows you to insert the cart icon using shortcode [enquiry_cart_icon]

= I want to disable the Subject field in the enquiry form =
Pro version allows you to disable form fields. You can disable any form field except Name and Email as those are needed for the internal working of the plugin

= I want to change the sequence of the form fields on the enquiry form =
In the Pro version you can change the position of the form field elements by changing the order value. The element with the lowest value will be at the top and the one with the highest value will be at the bottom. The position of the Terms and conditions and Submit button can't be changed; they will remain at the bottom

= I want to disable enquiry for a particular product category =
In the pro version you can disable enquiry for a category, so all the products in that category will not have the enquiry option

= I only want to enable the enquiry on a particular category of products =
In the Pro version you can do this. You will have to disable the enquiry on all the categories and just keep it enabled for the particular categories where you want to show the enquiry

= I want to show enquiry to users with specific roles only =
You can do that in the pro version

= I want to hide the Add to cart button only for the products for which enquiry is enabled =
You can do that in the pro version. It allows you to hide the Add to cart for all products, or hide it for the products for which you have enabled the enquiry option

= I want to disable the Add to cart button for specific products =
Yes, you can do that in the pro version. You can disable Add to cart for specific products or categories of products

= I want to make the Enquiry button smaller in size =
You can set the width of the button from the Enquiry button settings tab; there you can specify the width of the button in PX and you can also change the font size of the button as well 

= Disable message box next to the product in enquiry cart =
You can do that in the pro version

= Customer can see their old submitted enquiries =
In the pro version, if a user submits the enquiry while logged in to the account then they will be able to access those enquiries under the My Enquiry section (which will be located in the My Account section of WooCommerce) 

= I want to show country dropdown in the enquiry form =
Pro version supports country dropdown; it will show the countries based on your setting in WooCommerce > Settings > General (Selling location(s))

= Can I add extra fields in the enquiry form =
In the PRO version you can add extra field types like Text, Textarea, Dropdown, Radio input field 

= Can I show product total in the enquiry cart =
In the pro version you can show the total amount of the product in the enquiry cart

= Can I give image upload option in the enquiry form =
In the Pro version you can add an image upload option in the enquiry form

= Can I create an add to enquiry URL = 
Yes, you can do that in the pro version, e.g.: https://www.yourwebsite.com/?add-to-enquiry=product_id 

= Can I create add to enquiry URL for the variable product =
Yes, in the pro version you can create that, e.g.: https://www.yourwebsite.com/?add-to-enquiry=product_id&variation_id=var_id (replace product_id and var_id with the respective IDs)

= Images are not showing in the enquiry email =
Go to the Email settings tab in our plugin and then enable this option "Enable this option if product and logo images are not shown in the email or you don't want images attached in email"

= Product image and logo are going as attachments in the email =
By default the plugin sends the image as an attachment as it helps to avoid the spam filter. If you want to add images by URL and not as attachments then go to the Email settings tab and enable this option "Enable this option if product and logo images are not shown in the email or you don't want images attached in email"

= Is it HPOS compatible =
Yes, the Free version and PRO version both are HPOS compatible

= Two-column layout for enquiry form =
Pro version has a setting using which you can make the enquiry form a two-column layout. In the free version you can achieve that using CSS code 

= Can I send the enquiry data to Zapier in Free version =
Yes. Go to the Advanced options tab and there you can insert your webhook URL, so when the enquiry is submitted the data will be sent to that webhook URL in JSON format (Zapier, Pabbly Connect, Microsoft Power Automate, Workato, Tray.io, etc)

= Is there anything for spam protection =
Protect your enquiry form from spam by using CAPTCHA, which requires users to complete a simple test, or a honeypot, an invisible field that bots will fill but humans will ignore. Both methods help reduce automated submissions, ensuring only real users can submit the form.

= Get WooCommerce Product Enquiry in Telegram =
Yes, in the pro version you can get the enquiry in Telegram as well. You will have to set your Telegram bot token and chat ID in the plugin settings page; once you do that all the enquiries will be sent to your Telegram account as well
